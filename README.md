SIPAC-B es una solución tecnológica integral de GovTech diseñada para modernizar, asegurar y transparentar los procesos democráticos, consultas vecinales y presupuestos participativos.

Al descentralizar el registro de votos mediante tecnología Blockchain e integrar motores de Inteligencia Artificial (NLP/Big Data), el sistema erradica la opacidad electoral, reduce la fricción de participación y transforma los votos en inteligencia gubernamental accionable.

    Características Principales:

        • Identidad Híbrida Segura (Sin brecha digital): Validación de ciudadanos mediante el escaneo de credenciales oficiales (OCR) combinado con autenticación de doble factor vía SMS (2FA).

        • Inmutabilidad Criptográfica: Cada voto es cifrado (AES-256) desde el dispositivo del usuario y registrado como una transacción inmutable en una red Blockchain permisionada.

        • Transparencia Activa (Caja de Cristal): Incluye un "Portal del Observador" público para que ONGs y auditores visualicen la confirmación de bloques en tiempo real mediante WebSockets.

        • Inteligencia de Datos (Big Data): Motor analítico capaz de procesar propuestas masivas, categorizarlas por sentimiento y generar Mapas de Calor geoespaciales para la toma de decisiones.

        • Gestión Ágil (Drag-and-Drop): Panel administrativo intuitivo que permite a los funcionarios configurar boletas y elecciones en minutos sin necesidad de escribir código.

    Stack Tecnológico (Tech Stack)

    El proyecto utiliza una arquitectura de microservicios dividida en tres pilares principales:

        • Frontend (Capa de Presentación): HTML5, Tailwind CSS, JavaScript Vanilla / React.js (Diseño UX "No Crypto").

        • Backend Core (Lógica Transaccional): Node.js con Express. Integración con APIs de terceros (Twilio SMS, OCR) y conexión a Blockchain.

        • Backend Analítico (Motor de Inteligencia): Python con FastAPI. Uso de librerías de Procesamiento de Lenguaje Natural (NLP) y análisis de datos.

        • Bases de Datos: MongoDB (Datos operativos y Data Lake).

        • Blockchain: Hyperledger Fabric (o red DLT equivalente).
